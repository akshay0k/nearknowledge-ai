# NearKnowledge AI

NearKnowledge AI is a modern AI-powered document intelligence platform that enables users to upload PDF documents, extract and organize knowledge, and interact with their files through natural language conversations powered by Google Gemini AI.

The platform combines document management with artificial intelligence to create a seamless knowledge assistant for students, developers, researchers, and professionals.

## ✨ Features

- 🔐 JWT Authentication
- 📄 PDF Upload & Management
- 🤖 AI Chat with Documents (Google Gemini)
- 📚 Smart Knowledge Library
- 💬 Interactive AI Workspace
- 🔍 Text Extraction & Chunk Processing
- 📱 Responsive Modern UI
- 🌙 Dark Theme
- ⚡ Fast React + Vite Frontend
- 🗄️ MongoDB Database
